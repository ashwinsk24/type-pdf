# List Edge Cases

## Multi-paragraph list items

- First paragraph of item one.

  Second paragraph of item one with **bold text** and *italic text*.

- First paragraph of item two.

  Second paragraph of item two with `inline code`.

  Third paragraph of item two with ~~strikethrough~~.

- Single paragraph item to test spacing consistency.

## List items with code blocks

- Item with a code block below:

  ```
  code inside list item
  with multiple lines
  ```

- Another item after code list item
  - Nested item with `inline code`

## List items with blockquotes

- Outer list item

  > Blockquote inside a list item
  > with **bold** and *italic*

- Next list item after blockquote

## List items with links

- Click [this link](https://example.com) for more information
- Or visit [another page](https://example.org) with **bold text**
- Multiple [links](https://example.com) in [one](https://example.org) item

## Empty list items

- 
- Item with content after empty item
- 
  - Nested item after nested empty
